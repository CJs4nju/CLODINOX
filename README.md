# Calculadora de Grades Metalon - CLODINOX

Ferramenta profissional desenvolvida para a **CLODINOX - Soluções em Metais**, otimizada para cálculos técnicos de serralheria.

## 🛠️ Funcionalidades

- **Cálculo de Espaçamento:** Define o vão exato entre os metalons.
- **Cálculo Inverso com Folga:** Define a quantidade ideal de metalons e mostra a **folga total** resultante no vão.
- **Design Premium:** Identidade visual da CLODINOX (Cores escuras, vermelho e prata).
- **Multi-plataforma:** Versões otimizadas para **Mobile** (index.html) e **Desktop** (desktop.html).

## 📐 Lógica Técnica

A calculadora utiliza a fórmula:
`Espaçamento = (Medida Total - Soma das Laterais - (Face do Metalon * Quantidade)) / (Quantidade + 1)`

No modo inverso, o sistema calcula a quantidade exata, arredonda para o valor inteiro mais próximo e informa a **Folga Total** (diferença em mm no vão) para ajuste fino na montagem.
